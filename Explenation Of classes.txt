- **Vehicle.java**  
  - Abstract class (satisfies abstract/interface requirement).  
  - Defines `revEngine()` and `race()` → forces polymorphism.  

- **Car.java**  
  - Demonstrates **multilevel inheritance** (extends Vehicle).  
  - Adds attribute `topSpeed`.  

- **DodgeChallenger.java**  
  - Another level in multilevel inheritance.  
  - Base class for specific Challenger models.  

- **ChallengerSXT.java, ChallengerRT.java, ChallengerHellcat.java, ChallengerDemon.java**  
  - Demonstrate **hierarchical inheritance** (all extend DodgeChallenger).  
  - Each overrides `revEngine()` (polymorphism).  

- **DodgeChallengerGame.java**  
  - Contains **loops** (game menu).  
  - Uses **randomness** (dice rolls + computer car).  
  - Provides **interaction** (user choices).  
  - Implements quarter-mile race mechanic.  

- **Main.java**  
  - Entry point → starts the game.
